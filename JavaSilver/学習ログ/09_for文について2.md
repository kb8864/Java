# 学習内容



> [!WARNING]
> 条件式ではカンマ（,）は使えません。❌< 3, j < 5;
> 
> もし「両方の条件を満たす間」としたいのであれば、i < 3 && j < 5 のように論理演算子を使う
> 
> 実行順序: for 文の更新式（右端）は、「本体の { } の処理がすべて終わった後」 に実行されるので、一度for文の中の処理が終わったら更新式が動く
<details><summary>間違いJava</summary>

```
package sample;

public class ControlStructures5 {
  public static void main(String[] args) {
    for (int i = 0, j = 0; i < 3, j < 5; ❸i++) {//ここでコンパイルエラー
      System.out.println(i++);❶
      j += i;❷
    }

  }
}

```

</details>

<details><summary>正しいJava</summary>

```package sample;

public class Main {
    public static void main(String[] args) {
        // 条件式を i < 3 && j < 5 に修正
        for (int i = 0, j = 0; i < 3 && j < 5; i++) {
            System.out.println(i++); // 本体でのインクリメント
            j += i;
        }
    }
}

```

</details>


> [!TIP]
> ２重ループ の例
>
> ２重ループ ならまず外側のループは無視する
>
> 内側のループがわかったらそれを置き換える、それを外側のループで考える
</details>
