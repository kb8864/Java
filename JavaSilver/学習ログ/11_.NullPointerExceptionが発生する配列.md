> [!WARNING]
> やってしまった勘違い
>
> Item[] items = new Item[3];これを中身のあるItem オブジェクトが３つ作られたと誤判断した
>
>  正確には空の箱が３つ作られただけ
> 
> 現実: items = [null, null, null]
>
> items[i].price の実行:すると最初の箱にはnullが入っており、もうここでNullPointerExceptionの発生してしまうのでコンパイルエラー


<details><summary>間違ったjava</summary>

```
*package sample;

public class ArrayQa3 {
  public class Item{
    String name;
    int price = 100;
  }

  public static void main(String[] args) {
      Item[] items = new Item [3];⭐️
      int total = 0;
      for (int i = 0; i < items.length; i++){
        total += items[i].price;//コンパイルエラー
      }
    System.out.println(total);
  }
}

```

</details>

> [!TIP]
> price は 100 で固定される
>
> 変数 i（0, 1, 2...）は、あくまで「何番目の要素か」を示す**住所（インデックス）**であり、値段とは何の関係もありません
>
> 中身を使うには: items[0] = new Item(); のように、一つずつ実体を作って代入する必要がある。


<details><summary>正しいjava</summary>

```

// Itemクラス（
class Item {
  String name;
  int price = 100; // 価格は100円で固定
}

public class ArrayQa3 {
  public static void main(String[] args) {
    // 1. まず「枠」を3つ作る（中身はまだ null）
    Item[] items = new Item[3];

    // 【ここが修正ポイント！】
    // 2. 中身（実体）を作って、一つずつ箱に入れる
    items[0] = new Item(); // 0番目にItemを入れる
    items[1] = new Item(); // 1番目にItemを入れる
    items[2] = new Item(); // 2番目にItemを入れる

    // 3. 計算用の変数
    int total = 0;

    // 4. ループで合計を計算
    for (int i = 0; i < items.length; i++) {
      // items[i]には実体が入っているのでエラーにならない！
      // items[i].price は常に 100 です
      total += items[i].price;

      // トレース用に出力
      System.out.println(i + "番目の値段: " + items[i].price);
    }

    // 5. 結果表示
    System.out.println("合計: " + total);
  }


```　
</details>
