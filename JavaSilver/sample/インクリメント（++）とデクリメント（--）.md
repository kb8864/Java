# インクリメント（++）とデクリメント（--）

## それぞれの違い
- 前置 (++a): 変数を更新してから、その値を式で使用する

- 後置 (a++): 変数の現在の値を式で使用してから、変数を更新する

```

package sample;

public class Prc1 {

  // mainメソッドは「実行の司令塔」として使います
  public static void main(String[] args) {

    System.out.println("\n=== 問題2 ===");

  public static void solveQ1() {

              //b = 32

        c = 12, d = 22

  public static  void solveQ4() {
    int a = 10;

    if (a++ > 10 && ++b > 10) {
      System.out.println("True");
    }
    System.out.println(a + ", " + b);
  }
```

# 問題4について

論理積 && の左側が false の場合、右側は評価されないことを忘れていた。
左側を確認後、右側も確認してしまって
10、11と回答してしまってたため間違い

左側 a++ > 10: 後置なので 10 > 10 と評価される。これは false。
この時点で、a は 11 
&& の左が false なので、式全体の結果が false で確定。右側の ++b > 10 は実行されない

正解は11, 10
